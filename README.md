# 下雪特效

一个基于 Electron 开发的 Windows 下雪特效应用，为您的桌面带来冬日雪景的浪漫氛围。

## 功能特点

- ❄️ 逼真的雪花飘落效果
- 🎨 可自定义雪花大小、数量、速度、积雪高度和积雪速度
- 🖱️ 鼠标移动，可带动雪花或清除积雪
- 🔧 简单易用的设置界面
- 📱 支持多显示器

## 安装要求

- Windows 10/11
- Node.js 18.x 或更高版本
- ⚠️ **资源占用说明**  
   - **CPU占用**：★★★★★（极高）  
   - **内存占用**：★★★★★（极高）  
   - **储存占用**：★★★★★（极高）

## 使用说明

### 基本操作

1. **启动应用**：从桌面快捷方式或开始菜单启动
3. **自定义效果**：在设置界面中调整雪花参数
4. **清除积雪**：移动鼠标到积雪区域，积雪会被清除
5. **退出应用**：右键点击托盘图标，选择"退出"

### 设置选项

- **雪花大小**：调整雪花的直径（2-20px）
- **雪花数量**：控制屏幕上雪花的数量（50-500个）
- **下落速度**：调整雪花飘落的速度（1-5倍）
- **积雪高度**：控制地面积雪的最大高度（0-3000px）
